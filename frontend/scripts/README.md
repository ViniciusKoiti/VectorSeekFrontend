# Scripts utilitários do backlog

Este diretório contém scripts auxiliares para facilitar o acompanhamento das notas do vault Obsidian.

## `start_activity.py`

Marca uma atividade como iniciada alterando o `status` para `in-progress` e registrando o timestamp de início na frontmatter.

```bash
./frontend/scripts/start_activity.py E1-A1-1
```

Use `--dry-run` para visualizar o resultado antes de gravar as alterações.
