# E1-A3 · Construir app shell responsivo

## Descrição técnica
- Criar `AppShellLayoutComponent` em `apps/platform/src/app/layout` usando `mat-sidenav-container` com suporte a dark mode.
- Implementar `SideNavComponent`, `TopBarComponent` e `QuotaSummaryComponent` como componentes standalone reutilizáveis.
- Integrar Tailwind CSS com tokens de design centralizados (`tailwind.config.js`) para suportar temas claro/escuro e customização de logotipos.
- Garantir SSR-friendly com `@angular/cdk/layout` para detectar breakpoint sem acessar `window` diretamente.
- Criar histórias no Storybook (`libs/ui`) demonstrando variações de layout.

## Critérios de aceite

- Layout responde a breakpoints `lg`, `md`, `sm`, colapsando menu em mobile.
- Quotas exibem dados mockados provenientes de `QuotaSummaryComponent` com inputs tipados.
