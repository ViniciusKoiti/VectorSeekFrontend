## Como usar
1. Instale o Obsidian e crie/abra um vault local.
2. Copie a pasta `docs/frontend/obsidian` para dentro do vault.
3. Utilize o `Graph View` para visualizar dependências com base no campo `dependencies` das notas.
4. Atualize os checklists conforme as atividades avancem; o Obsidian mantém o histórico de marcações.
