# E7-A1 · Implementar wizard de setup operacional

## Descrição técnica
- Criar `SetupModule` com rota `/app/setup` e `SetupWizardComponent` utilizando `CdkStepper` para etapas personalizadas (Ambiente, Banco, Serviços, Resumo).
- Carregar configurações iniciais via `/api/setup/status` populando formulário com dados existentes.
- Implementar componentes de etapa com validações e mensagens traduzidas (`@ngx-translate/core` namespace `setup`).
- Garantir que wizard funcione com SSR e suporte reentrada (retomar progresso salvo).
- Documentar fluxo no Storybook para onboarding interno.
